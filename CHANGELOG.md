# Changelog

## [0.2.0] - 2025-11-10

### Added
- Remote-source fetcher feature with support for git repositories and archives.
- `--fetch` and `--source-url` CLI flags to enable remote scanning.
- GUI integration for the remote-source fetcher.
- License verification for remote sources.
- Unit tests for the fetcher module.

## [0.1.0] - 2025-11-10

### Added
- Initial release of secret-hunter.
- Core scanning logic with rule-based regex matching.
- CLI and GUI interfaces.
- Authorization check to ensure responsible use.
- JSON report generation.
- Unit tests for the scanner module.
